# SuperDucky.Connector

## 功能

此模块自动导出所有游戏物品并实时追踪您的收集进度。

### 物品百科
自动导出游戏中所有物品的完整数据，包括：
- 物品名称和描述
- 物品图标和预览图片
- 详细物品属性（可堆叠、重量、价格、品质、标签、属性值）

### 获取追踪
追踪您在游戏中收集的物品：
- 实时监控您的背包库存
- 记录物品获取事件和时间戳
- 支持多个存档槽位，各自独立追踪

### 工作方式
- 游戏加载时自动运行
- 将数据保存到系统临时目录
- 收集新物品时持续更新
- 无需手动配置

### 数据位置
所有导出的数据保存在系统临时目录下的 `SuperDucky.Connector` 文件夹中：
- **Windows**: `C:\Users\{username}\AppData\Local\Temp\SuperDucky.Connector\`
- **Linux/macOS**: `/tmp/SuperDucky.Connector/`

外部工具可以读取这些数据来显示物品信息、分析您的收集进度，或与其他应用程序集成。
